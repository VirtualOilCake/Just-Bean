# 黄豆而已 Just Bean
  这个项目的起源完全是因为一天我看到了有人说`文明用语从使用脱水黄豆开始😄👍`，我觉得很有道理于是便有了这个项目（    
  最早的目的只是为了研究一下安卓的Animation组件，结果不知不觉写了好多有的没的东西，差不多得了😅 
## 功能
 - 你可以点击黄豆，并且让黄豆跳起来。
 - 你可以让黄豆跳起来后头上的汗珠消失。
 - 你可以用它浪费你一整天的时间，甚至更多，就像我一样。
 - 支持多语言和黑色模式。
